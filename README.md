## Steps

install npm 
```
apt install npm
```
install node js

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
source ~/.bashrc
nvm install 'lts/*'
nvm use 'lts/*'
```

### 1. install screnn

```bash
apt install screen 
```
```bash
screen -S sonicbot
```
### 2. clone Repository

```bash
git clone https://github.com/dlzvy/SONIC-BOT.git
```

```
cd SONIC-BOT
```

### 3. install Dependency

```bash
npm i
```

### 4. import private key di  account.js

```bash
nano account.js
```

### 5. start bot

```bash
npm start
```
