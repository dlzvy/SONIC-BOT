
## Steps

### 1. Clone Repository

```bash
git clone https://github.com/dlzvy/SONIC-BOT.git
```

```
cd SONIC-BOT
```

### 2. Instal Dependency

```bash
npm i
```

### 3. import private key di  account.js

```bash
nano account.js
```

### 4. start bot

```bash
npm start

```

### 5. Run script

```bash
npm run start
```
